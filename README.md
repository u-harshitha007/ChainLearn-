# blockchain mini project 

A hands-on blockchain learning project built from scratch using Python to help students understand how blockchains work through practical implementation.

## Learning

- SHA-256 Hashing
- Blocks & Block Structure
- Blockchain Architecture
- Previous Hash Linking
- Tamper Detection
- Chain Validation
- Cryptographic Integrity

## Project Structure

- `backend/` – Python blockchain engine and API server.
- `frontend/` – Interactive interface for visualizing blockchain concepts.

## Goal

ChainLearn focuses on learning blockchain fundamentals by building each component step-by-step rather than using existing blockchain frameworks.
